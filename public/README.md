# What-If App Downloads

This directory contains the mobile app installers for What-If.

## Files

- `what-if-android.apk` - Android application (place your APK file here)
- `what-if-ios.ipa` - iOS application (place your IPA file here)

## Instructions

1. Replace these placeholder files with your actual app files
2. For production, consider using app store links instead:
   - Android: Google Play Store URL
   - iOS: Apple App Store URL

## Alternative Implementation

You can also use external hosting services like:
- Firebase App Distribution
- TestFlight (for iOS beta testing)
- Direct APK hosting for Android
